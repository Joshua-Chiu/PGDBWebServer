About CSV files

A CSV file is a text file that is formatted as a table. Each line has data that is separated from each other by commas. Because a CSV file is only text, it can be used with any text editor on any operating system. To open a CSV file, you can use a spreadsheet program such as Microsoft Excel or LibreOffice.

Best practices if your CSV files are raising an error

Download the template files from the links below or your dashboard and open the file in Microsoft Excel. If Excel warns you about potential data loss from the comma delimiter, please ignore the prompt. If you have the data for import already formatted in another Excel document, open it and copy the contents into the CSV. Ensure that the column label corresponds to the correct column of data. Save the file (Be careful not to save it as a .xls or .xlsx file) and upload it to the database. If an the same error is continually raised, please see the reporting an error section of this page.
